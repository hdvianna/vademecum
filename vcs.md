### Tag
Creating a tag:

`>git tag <tag name>` 

Example: `$git tag 1.1.10.rc` 

Pushing a tag:

`>git push --tags`

### Starting a Repository

```
>git init
>git add README.md
>git commit -m "first commit"
>git remote add origin <repository-url>
>git push -u origin master
```
